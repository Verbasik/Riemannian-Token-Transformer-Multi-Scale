## 🔍 КОРНЕВЫЕ ПРИЧИНЫ ДЕГРАДАЦИИ

### 1. Семантическая путаница метаклассов
**Проблема**: Метаклассы не являются взаимоисключающими
- "DAILY_LIFE" семантически пересекается со ВСЕМИ остальными категориями
- "SOCIAL_INTERACTION" ↔ "ENTERTAINMENT_LEISURE" трудноразличимы
- Граница между "COMMERCE_SERVICES" и "WORK_EDUCATION" размыта

**Доказательство**:
- Класс 1 (DAILY_LIFE): почти равномерное распределение путаниц между классами 0, 3, 5, 6, 7
- Confusion matrix показывает высокую симметричную путаницу между классами 0 и 7

### 2. Дисбаланс классов (несмотря на CB-Focal)
**Распределение валидационной выборки**:
- Макс: класс 6 (231 образцов)
- Мин: класс 3 (75 образцов)
- Соотношение: 3.08x

**Эффект**:
- Малые классы (2, 3) имеют низкий recall
- CB-Focal Loss не полностью компенсирует дисбаланс при такой семантической путанице

### 3. Недостаточная дискриминативная способность признаков
**Симптомы**:
- Средняя confidence 30% — модель не может уверенно разделить классы
- Высокая энтропия предсказаний — равномерное распределение вероятностей
- Даже лучшие классы (5, 7) имеют F1 < 0.45

**Возможные причины**:
- SPD-признаки (корреляции логарифмированных ковариаций) недостаточно дискриминативны для тонкой семантической дифференциации
- Требуется более богатая feature space (например, SPDNet для обучаемых SPD-преобразований)

### 4. Высокая внутриклассовая вариативность
**Наблюдение**: Val Loss > Train Loss (gap -0.21)

**Гипотезы**:
- Внутри каждого метакласса объединены разнородные оригинальные классы (например, метакласс 0 объединяет 6 оригинальных классов: 7, 9, 12, 24, 34, 38)
- ЭЭГ паттерны для разных концептов внутри одного метакласса могут быть очень разными

### 5. Неэффективность текущей архитектуры
**Сравнение**:
- Исторический лучший: F1 = 0.2915 (те же данные, та же модель)
- Текущий: F1 = 0.2627
- Деградация: -9.9%

**Возможные причины деградации**:
- Изменение пути к данным могло привести к другой загрузке/preprocessing
- Возможна разница в random seed или версии библиотек
- Требуется воспроизвести точные условия исторического запуска
