🔴 БЛОКЕРЫ ДЛЯ ФИНАЛИЗАЦИИ:

1. Subject-Aware Cross-Validation [HIGH]
   ├─ Текущее: StratifiedKFold (перемешивает субъектов)
   ├─ Требуется: StratifiedGroupKFold (группировка по subject_id)
   └─ Почему: иначе невозможно оценить cross-subject обобщаемость

2. Full K-Fold Evaluation [HIGH]
   ├─ Текущее: только Fold 1 для всех субъектов
   ├─ Требуется: все 5 folds для корректной статистики
   └─ Почему: одного fold недостаточно для доверительных выводов

🟡 ВАЖНЫЕ ПРОБЕЛЫ (MEDIUM приоритет):

3. Bootstrap Доверительные Интервалы
   └─ Требуется для: mean±CI по фолдам (например, 95% CI)

4. Статистические Тесты
   ├─ Wilcoxon signed-rank для сравнений
   └─ Benjamini–Hochberg для множественных сравнений

5. Калибровка вероятностей
   ├─ Temperature scaling (post-hoc)
   └─ Метрики: ECE, NLL, Brier score
