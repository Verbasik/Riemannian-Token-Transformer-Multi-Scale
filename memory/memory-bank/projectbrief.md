# Project Brief — EEG_to_Text

## Миссия проекта
Разработка системы декодирования **воображаемой речи** по сигналам ЭЭГ с использованием глубокого обучения и римановой геометрии.

## Цель
Создание модели классификации 8 мета-классов воображаемой речи из 39 исходных классов на основе многоканальных ЭЭГ-сигналов.

## Ключевые требования

### Функциональные
- **Архитектура модели**: Двухмасштабный Riemannian Token Transformer (RTTMultiScale)
- **Входные данные**: Предобработанные pkl-файлы с ЭЭГ-сигналами (125 каналов)
- **Выходные данные**: Классификация на 8 мета-классов
- **Нормализация**: Гибридная (subject-wise centering + global scaling)
- **Кросс-валидация**: Stratified K-Fold (5 splits)

### Технические
- **Python**: 3.12+
- **Фреймворк**: PyTorch 2.2+
- **Устройства**: CUDA / CPU (MPS fallback)
- **Ключевые библиотеки**: MNE, scikit-learn, NumPy, SciPy

### Метрики качества
- **Основная**: F1-macro (целевая ≥0.29)
- **Дополнительные**: Accuracy, Balanced Accuracy, Per-class Precision/Recall

## Текущий статус
**Phase 4B** — Стабильная версия с subject embeddings и гибридной нормализацией
