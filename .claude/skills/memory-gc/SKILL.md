## Шаги

### 1. Архивация старых сессий
Найди файлы в `memory-bank/.local/SESSIONS/` старше 30 дней → перенеси в `ARCHIVE/`.

### 2. Очистка старых архивов
Перечисли файлы в `ARCHIVE/` старше 90 дней. Спроси у пользователя — удалять ли.

### 3. Очистка заменённых решений
В `DECISIONS.md` найди строки со статусом ❌. Проверь, что заменяющий ADR существует.
Если подтверждено — предложи удалить заменённую строку (ADR-файл оставить для истории).

### 4. Сжатие CURRENT.md
Удали отмеченные пункты старше последнего handoff. Оставь только активные.
Убедись, что файл не превышает 80 строк.

### 5. Проверка temporal health
Для каждого стабильного файла проверь дату "Верифицировано".
Если >60 дней — добавь в рекомендации аудита.

### 6. Отчёт
Перечисли: архивировано, кандидаты на удаление, очищено, устаревшие файлы.
