---
description: Правила работы с файлами memory bank
globs: memory-bank/**
---

- Соблюдай лимиты строк из заголовка каждого файла.
- Используй ISO 8601: ГГГГ-ММ-ДД или ГГГГ-ММ-ДД ЧЧ:ММ.
- Каждый новый/обновлённый факт → "Верифицировано: ГГГГ-ММ-ДД".
- Один факт — один файл. Ссылка: `см. ARCHITECTURE.md § секция`.
- CURRENT.md: добавляй timestamped блоки, не перезаписывай чужие записи.
- Никогда: секреты, токены, пароли, PII, сырой вывод терминала.
- При добавлении решения — сначала DECISIONS.md, потом опционально ADR.

- Файлы уровня 🔒 (CONSTITUTION.md, ADR/*.md) — НИКОГДА не перезаписывай. Только предлагай изменения для ревью.
- Файлы уровня 🔐 — при обновлении обязательно ставь «Верифицировано: [дата]».
